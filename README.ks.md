# فرسٹ کنٹریبیوشنز – کشمیری ترجمہ

## پہلوٹ مدد

یہ پروجیکٹ نوآموز ڈیولپرن ہند خاطر آسان تِہ رہنمائی کرن والہ چھہ تاکہ تِم آسانی سان اپنی پہلی اوپن سورس کنٹریبیوشن کٔرِتھ ہیکن۔

اگر توہیہ تہِ اوپن سورس منہ پہلی وار حصہ لِوان چھیو، تِہ نیچے دِتھہ گٔیٲ قدم فالو کٔریو۔

## شرط

اگر توہیہ مشینس منہ Git انسٹال نئ چُھ، تِہ پہلے یہ انسٹال کٔریو:
https://docs.github.com/en/get-started/quickstart/set-up-git

# قدم بہ قدم رہنمائی

## 1. ریپوزیٹری فورک کٔریو

- اِس صفحس پیٹھ Fork بٹن پیٹھ کلک کٔریو  
- یُتھ کرن سان یہ ریپوزیٹری توہندِ گِٹ ہب اکاؤنٹس منہ کاپی بنی

## 2. ریپوزیٹری کلون کٔریو

ٹرمینل کھولیو تِہ یہ کمانڈ چلائیو:

git clone "توہندہ کاپی کرتھہ URL"

## 3. برانچ بنٲویو

cd first-contributions

git switch -c my-new-branch

## 4. تبدیلیاں کٔریو

Contributors.md فائل کھولیو  
اپنی نٲو لسٹہ منہ جمع کٔریو  
فائل محفوظ کٔریو

git add Contributors.md

git commit -m "Add my name"

## 5. تبدیلیاں پُش کٔریو

git push -u origin my-new-branch

## 6. پل ریکویسٹ جمع کٔریو

گِٹ ہبس پیٹھ واپس جائیو  
Compare & Pull Request بٹن استعمال کٔریو

# مبارک!

توہیہ کامیابی سان اپنی پہلی اوپن سورس کنٹریبیوشن مکمل کٔرو!

اوپن سورس دُنیاہ منہ خوش آمدید!
