# My Open Source Learning Plan

## Goal
Master the basics of open source contribution using the First Contributions repository.

## Tasks

### Phase 1: The Basics (The "Hello World")
- [x] **Step 1:** Create a new branch for my contribution (e.g., `add-my-name`).
- [x] **Step 2:** Add my name to `Contributors.md` to register my contribution.
- [x] **Step 3:** Commit the change with a clear message.
- [ ] **Step 4:** Push the branch to the remote repository.
- [ ] **Step 5:** (Manual) Go to GitHub and submit a Pull Request.

### Phase 2: Advanced Git Skills (Optional)
- [ ] **Squashing:** Practice combining multiple small commits into one clean commit.
- [ ] **Conflict Resolution:** Intentionally create a conflict and practice resolving it.
- [ ] **Rebasing:** Learn to rebase a branch to keep history clean.

### Phase 3: Knowledge Check
- [ ] Read `docs/additional-material/git_workflow_scenarios/gitflow.md` to understand branching models.
