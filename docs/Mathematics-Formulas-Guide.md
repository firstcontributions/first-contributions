# Mathematics Basic Formulas Guide

A comprehensive collection of essential mathematical formulas for students and beginners.

## 📐 Algebra Formulas

### Basic Identities
- **(a + b)² = a² + 2ab + b²**
- **(a - b)² = a² - 2ab + b²**
- **a² - b² = (a + b)(a - b)**
- **(a + b)³ = a³ + 3a²b + 3ab² + b³**

### Quadratic Formula
For equation: **ax² + bx + c = 0**
- **x = [-b ± √(b² - 4ac)] / 2a**

### Arithmetic Progression (AP)
- **nth term: aₙ = a + (n-1)d**
- **Sum of n terms: Sₙ = n/2 [2a + (n-1)d]**

## 📊 Geometry Formulas

### 2D Shapes
**Rectangle:**
- Area = length × width
- Perimeter = 2(length + width)

**Circle:**
- Area = πr²
- Circumference = 2πr

**Triangle:**
- Area = ½ × base × height
- Pythagorean Theorem: a² + b² = c²

### 3D Shapes
**Cube:**
- Volume = a³
- Surface Area = 6a²

**Sphere:**
- Volume = (4/3)πr³
- Surface Area = 4πr²

**Cylinder:**
- Volume = πr²h
- Curved Surface Area = 2πrh

## 📈 Trigonometry Formulas

### Basic Ratios
- **sin θ = Opposite/Hypotenuse**
- **cos θ = Adjacent/Hypotenuse**
- **tan θ = Opposite/Adjacent**

### Important Identities
- **sin²θ + cos²θ = 1**
- **1 + tan²θ = sec²θ**
- **1 + cot²θ = cosec²θ**

### Angle Formulas
- **sin(A+B) = sinA cosB + cosA sinB**
- **cos(A+B) = cosA cosB - sinA sinB**

## 🧮 Calculus Formulas

### Differentiation
- **d/dx(xⁿ) = nxⁿ⁻¹**
- **d/dx(sin x) = cos x**
- **d/dx(cos x) = -sin x**
- **d/dx(eˣ) = eˣ**

### Integration
- **∫xⁿ dx = xⁿ⁺¹/(n+1) + C**
- **∫sin x dx = -cos x + C**
- **∫cos x dx = sin x + C**

## 📉 Statistics Formulas

### Mean, Median, Mode
- **Mean = Sum of observations / Number of observations**
- **Median = Middle value of sorted data**
- **Mode = Most frequent value**

### Standard Deviation
- **σ = √[Σ(x - μ)² / N]**

## 💰 Percentage Formulas
- **Percentage = (Value/Total) × 100**
- **Percentage Increase = [(New - Original)/Original] × 100**
- **Percentage Decrease = [(Original - New)/Original] × 100**

## 🔢 Number System
- **Prime Numbers:** 2, 3, 5, 7, 11, 13, ...
- **Even Numbers:** 2, 4, 6, 8, 10, ...
- **Odd Numbers:** 1, 3, 5, 7, 9, ...
