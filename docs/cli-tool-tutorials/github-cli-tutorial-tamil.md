# முதல் பங்களிப்புகள்

| Git Bash | Git Bash பதிப்பு |
| --- | --- |
| <img alt="Git Bash" src="https://cdn.icon-icons.com/icons2/2699/PNG/512/git_scm_logo_icon_170096.png" width="200"> | இந்தப் பக்கத்தில் Git Bash பயன்படுத்தி பங்களிக்கும் முறை விளக்கப்படுகிறது. |

முதன் முறையாக ஏதாவது செய்வது கடினம். குறிப்பாக மற்றவர்களுடன் இணைந்து வேலை செய்யும்போது தவறு செய்வது சங்கடமாக இருக்கும். ஆனால் ஓப்பன் சோர்ஸ் என்பதே ஒத்துழைப்பும் இணைந்து செய்யும் வேலையும் தான். புதிய திறந்த மூல பங்களிப்பாளர்கள் எளிதாகக் கற்றுக் கொள்ளவும் பங்களிக்கவும் இந்தத் திட்டம் உதவுகிறது.

கட்டுரைகளைப் படிப்பதும் டுடோரியல்களைப் பார்ப்பதும் உதவும். ஆனால் நேரடியாகச் செய்து பார்ப்பதே சிறந்தது. இந்தத் திட்டம் உங்களுக்கு வழிகாட்டுதலைத் தருகிறது. உங்கள் முதல் பங்களிப்பை எளிதாகச் செய்ய இது உதவும். நிதானமாகச் செய்தால் நன்றாகக் கற்றுக்கொள்வீர்கள். கீழே உள்ள வழிமுறைகளைப் பின்பற்றி உங்கள் முதல் பங்களிப்பைச் செய்யுங்கள். இது வேடிக்கையாக இருக்கும் என்று உறுதி அளிக்கிறோம்.

உங்கள் விண்டோஸ் கணினியில் Git Bash இல்லை என்றால், [அதை நிறுவவும்](https://git-scm.com/download/win).

<img align="right" width="300" src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/fork.png" alt="இந்த களஞ்சியத்தை பிரிக்கவும்" />

## இந்த களஞ்சியத்தை ஃபோர்க் செய்யவும்

இந்தப் பக்கத்தின் மேல் வலதுபுறத்தில் உள்ள Fork பொத்தானைக் கிளிக் செய்யவும். இது உங்கள் கணக்கில் இந்தக் களஞ்சியத்தின் நகலை உருவாக்கும்.

## களஞ்சியத்தை க்ளோன் செய்யவும்

இப்போது இந்த ரெப்போவை உங்கள் கணினியில் க்ளோன் செய்யவும்.

முக்கியம்: அசல் ரெப்போவை க்ளோன் செய்ய வேண்டாம். உங்கள் ஃபோர்க்கிற்குச் சென்று அதைக் க்ளோன் செய்யுங்கள்.

ரெப்போவை க்ளோன் செய்ய, "Code" பொத்தானைக் கிளிக் செய்து, கீழே உள்ள சரத்தை நகலெடுக்கவும்.

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-clone-1.png" alt="நகல் சரம்" />

நீங்கள் பதிவிறக்கிய Git Bash பயன்பாட்டைத் திறக்கவும். விண்டோஸ் கணினியில் கீழே உள்ள படம் போல் இருக்கும்.

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-terminal-1.png" alt="Git Bash டெர்மினலைத் திற" />

இந்தக் கட்டளையைப் பயன்படுத்தி இந்தத் திட்டத்தைச் சேமிக்க விரும்பும் கோப்புறைக்குச் செல்லவும்:

```bash
cd <folder-name>
```

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-terminal-2.png" alt="ஒரு கோப்புறைக்குள் செல்" />

மேலே நகலெடுத்த சரத்தைப் பயன்படுத்தி இந்தக் கட்டளையால் களஞ்சியத்தைக் க்ளோன் செய்யவும்:

```bash
git clone <repo-url>
```

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-clone-2.png" alt="களஞ்சியத்தைக் க்ளோன் செய்" />

மாற்றங்களைச் செய்ய, ரெப்போ இருக்கும் கோப்பகத்திற்குச் சென்று அதை VS Code-இல் திறக்கவும்.

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-terminal-3.png" alt="புதிதாகக் க்ளோன் செய்யப்பட்ட ரெப்போவில் செல்" />

## ஒரு கிளையை உருவாக்கவும்

இந்தக் கட்டளையைப் பயன்படுத்தி ஒரு கிளையை உருவாக்கவும். இது ஒரு கிளையை உருவாக்குவதோடு, உடனே அந்தக் கிளைக்கு உங்களை மாற்றவும் செய்யும்.

```bash
git checkout -b <branch-name>
```

உங்கள் கிளைக்கு உங்கள் பெயரைச் சேர்த்துப் பெயரிடவும். எடுத்துக்காட்டு: "add-james-smith".

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-branch.png" alt="ஒரு கிளையை உருவாக்கு" />

## தேவையான மாற்றங்களைச் செய்து அவற்றைக் காமிட் செய்யவும்

உரை திருத்தியில் `Contributors.md` கோப்பைத் திறக்கவும். பக்கத்தின் கீழே ஸ்க்ரோல் செய்யவும். அதில் உங்கள் பெயரைச் சேர்க்கவும். கோப்பைச் சேமிக்கவும்.

எடுத்துக்காட்டு: உங்கள் பெயர் James Smith என்றால், இப்படி இருக்க வேண்டும்:

[James Smith](https://github.com/jamessmith)

இந்தக் கட்டளையை இயக்கினால் `Contributors.md`-இல் மாற்றங்கள் இருப்பதைக் காணலாம்:

```bash
git status
```

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-status.png" alt="நிலையைச் சரிபார்" />

இப்போது அந்த மாற்றங்களைக் காமிட் செய்யுங்கள்.

முதலில், நீங்கள் செய்த மாற்றத்தை staging பகுதியில் சேர்க்கவும்:

```bash
git add <file-name>
```

இந்தக் கட்டளையால் ஒரு காமிட் செய்தியை எழுதவும்:

```bash
git commit -m "Add <your-name> to Contributors list"
```

`<your-name>` என்பதை உங்கள் பெயருடன் மாற்றவும்.

<img src="https://firstcontributions.github.io/assets/cli-tool-tutorials/git-bash-windows-tutorial/gb-commit.png" alt="மாற்றங்களைக் காமிட் செய்" />

உங்கள் காமிட் வெற்றியடைந்ததா எனப் பார்க்க, `git log --oneline` கட்டளையை இயக்கலாம்.
