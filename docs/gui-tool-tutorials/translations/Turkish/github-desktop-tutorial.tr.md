# First Contributions

| <img alt="GitHub Desktop" src="https://desktop.github.com/images/desktop-icon.svg" width="200"> | GitHub Desktop Edition |
| ----------------------------------------------------------------------------------------------- | ---------------------- |

Zor gelir. Herhangi bir işi ilk kez yapmak daima zor gelir. Özellikle başkalarıyla ortak çalışıyorsanız, hata yapmak içinize sinmez. Ancak "Açık Kaynağın" temelinde işbirliği ve birlikte çalışma yatmakta. Biz, açık kaynak projelere ilk kez katkıda bulunacak kişilerin bu süreci öğrenmesini ve ilk katkılarını sunmalarını kolaylaştırmayı istiyoruz.

Makale okumak ve eğitim videoları izlemek yardımcı olabilir, fakat bir işi gerçekten yapmanın yerini ne tutabilir ki? Bu proje yeni başlayanların veya ilk defa katkıda bulunacakların işini kolaylaştırmak ve onlara rehberlik etmek amacındadır. Unutmayın ki ne kadar rahat olursanız o kadar rahat öğrenirsiniz. Eğer bir GitHub projesine ilk defa katkıda bulunacaksanız, aşağıda gösterilen basit adımları takip etmeniz yeterli olacaktır. Söz veriyoruz, eğlenceli olacak.

Eğer bilgisayarınızda GitHub Desktop kurulu değil ise, [ yükleyin ](https://desktop.github.com/).

Eğer GitHub Desktop'un önceki bir versiyonunu kullanıyorsanız [ bu öğreticiye gidin ](github-desktop-old-version-tutorial.md).

<img align="right" width="300" src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/fork.png" alt="Bu depoyu forklayın" />

## Projeyi "forklama"

Sayfanın sağ üst köşesinde bulunan "Fork" (Çatalla) düğmesine tıklayarak bu projeyi çatallayın.
Bu işlem sizin hesabınız altında projenin bir kopyasını oluşturacaktır.

## Depoyu (Repository) klonlama

Şimdi bu depoyu bilgisayarınıza klonlayın.

ÖNEMLİ: ORİJİNAL DEPOYU KLONLAMAYIN. Kendi çatalınıza gidip oradan klonlayın.

Depoyu klonlamak için "Code" (Kod) düğmesine tıklayın, sonra "Open in GitHub Desktop" (GitHub Desktop'ta aç) seçeneğine tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-clonetodesktop.png" alt="Depoyu (Repository) klonlama" />

Bir açılır pencere açılacaktır. "Open GitHubDesktop" (GitHubDesktop.exe'yi aç) seçeneğini seçin.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-open-githubdesktop.png" alt="Depoyu (Repository) klonlama" />

"Open GitHubDesktop" seçeneğine tıkladıktan sonra "Clone a repository" (Bir Depoyu Klonla) diyalog kutusunu göreceksiniz. "Clone" (Klonla) düğmesine tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/clone-repository.png" alt="Depoyu (Repository) klonlama" height="400" />

Sonra, "How are you planning to use this fork?" (Bu çatalı ne amaçla kullanmayı düşünüyorsunuz) diye soran bir diğer diyalog kutusu açılacak. "To contribute to the parent project" (Ana projeye katkıda bulunmak için) seçeneğini seçip "Continue" (Devam) düğmesine tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/repository-clone-purpose.png" alt="Depo kullanımı" height="500" />

Artık GitHub'daki "first-contributions" deponuzu bilgisayarınıza kopyalamış oldunuz.

## Dal (Branch) oluşturma

Yukarıdaki "Current Branch" (Geçerli Dal) ikonuna tıklayıp "New Branch" (Yeni Dal) seçeneğine tıklayarak bir dal oluşturun.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch.png" alt="Dal oluşturun" />

Dalınıza isim verin <isminiz-eklendi>. Örneğin, "ahmet-yilmaz-eklendi"

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch-name.png" alt="Dalınıza isim verin" />

`Create branch` (Dal oluştur) düğmesine tıklayın.

## Gerekli değişiklikleri yapma ve değişiklikleri onaylama

Şimdi, History (Geçmiş) sekmesine gidin ve `Contributors.md` dosyasını sağ tıklayıp bir metin düzenleyicide açın. Sayfanın en altına inip isminizi ekleyin, ardından dosyayı kaydedin.

Örnek: İsminiz Ahmet Yılmaz ise şu şekilde görünmelidir.

\[Ahmet Yılmaz](https://github.com/ahmet-yilmaz)

Contributors.md dosyasında değişiklikler olduğunu ve bu değişikliklerin GitHub Desktop'a eklendiğini görebilirsiniz.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-status.png" alt="değişiklikleri kontrol edin" />

Şimdi bu değişiklikleri onaylayın (commit):

_summary_ (özet) alanına "Add `<isminiz>` to Contributors list" mesajını yazın.

`<isminiz>` yerine kendi isminizi yazın.

`Commit to <isminiz-eklendi>` (isminiz-eklendi için onayla) yazan düğmeye tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit1.png" alt="değişikliklerinizi onaylayın" />

En altta commit'in oluşturulduğunu görebilirsiniz.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit2.png" alt="değişikliklerinizi onaylayın" />

## Değişiklikleri GitHub'a gönderme

File -> Options (Dosya -> Seçenekler) yoluna tıklayın ve GitHub.com hesabınızla oturum açın. GitHub kullanıcı adınızı ve parolanızı girin.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-sign-in.png" alt="GitHub'a giriş yapın" />

Sağ üstteki `Publish` (Yayınla) düğmesine tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-publish1.png" alt="değişikliklerinizi gönderin" />

## Değişikliklerinizi inceleme için gönderme

GitHub'daki deponuza giderseniz `Compare & pull request` (Karşılaştır ve çekme isteği) düğmesini göreceksiniz. Bu düğmeye tıklayın.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/compare-and-pull.png" alt="çekme isteği oluşturun" />

Şimdi çekme isteğini (pull request) gönderin.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/submit-pull-request.png" alt="çekme isteğini gönderin" />

Yakında tüm değişikliklerinizi bu projenin master (ana) dalıyla birleştireceğim. Değişiklikler birleştirildiğinde bir bildirim e-postası alacaksınız.

## Bundan sonra ne yapabilirim?

Tebrikler! Katkıda bulunan biri olarak sıkça karşılaşacağınız standart _fork -> clone -> edit -> PR_ iş akışını tamamladınız!

Katkınızı kutlayın ve [web uygulamasına](https://firstcontributions.github.io#social-share) giderek bunu arkadaşlarınız ve takipçilerinizle paylaşın.
