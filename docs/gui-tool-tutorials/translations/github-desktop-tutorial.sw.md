# Michango ya Kwanza

  GitHub Desktop   Toleo la GitHub Desktop
  ---------------- -------------------------

Ni vigumu. Daima huwa vigumu mara ya kwanza unapofanya jambo fulani.
Hasa unaposhirikiana na wengine, kufanya makosa si jambo linalokufanya
ujisikie vizuri. Lakini open source inahusu ushirikiano na kufanya kazi
pamoja. Tulitaka kurahisisha namna wachangiaji wapya wa open source
wanavyojifunza na kuchangia kwa mara ya kwanza.

Kusoma makala na kuangalia mafunzo (tutorials) kunaweza kusaidia, lakini
hakuna kilicho bora zaidi kuliko kufanya jambo lenyewe bila kuharibu
chochote. Mradi huu unalenga kutoa mwongozo na kurahisisha namna
wanaoanza wanavyofanya mchango wao wa kwanza. Kumbuka, kadri unavyokuwa
mtulivu ndivyo unavyojifunza vizuri zaidi. Ikiwa unatafuta kufanya
mchango wako wa kwanza, fuata tu hatua rahisi zilizo hapa chini.
Tunakuahidi, itakuwa ya kufurahisha.

Ikiwa huna GitHub Desktop kwenye kompyuta yako,
[isakinishe](https://desktop.github.com/).

Ikiwa unatumia toleo la GitHub Desktop kabla ya 1.0, rejea mafunzo ya
toleo la zamani.

------------------------------------------------------------------------

## Fanya Fork ya Hifadhi hii

Fanya fork ya hifadhi hii kwa kubonyeza kitufe cha **fork** kilicho juu
upande wa kulia wa ukurasa huu.\
Hii itaunda nakala ya hifadhi hii kwenye akaunti yako.

------------------------------------------------------------------------

## Clone hifadhi

Sasa clone hifadhi hii kwenda kwenye kompyuta yako.

**MUHIMU:** USICLONE HIFADHI YA ASILI. Nenda kwenye fork yako na uiclone
hiyo.

Ili kuiclone, bonyeza "Code" kisha bonyeza "Open in GitHub Desktop".\
Bonyeza "Clone" kwenye kisanduku cha mazungumzo kinachoonekana.

Baada ya hapo chagua "To contribute to the parent project" kisha bonyeza
"Continue".

Sasa umenakili maudhui ya hifadhi ya first-contributions kutoka GitHub
kwenda kwenye kompyuta yako.

------------------------------------------------------------------------

## Unda Tawi (Branch)

Bonyeza "Current branch" juu kisha bonyeza "New branch".\
Ipe branch yako jina `<ongeza-jina-lako>` mfano `add-james-smith`.

Bonyeza `Create branch`.

------------------------------------------------------------------------

## Fanya mabadiliko na uyahifadhi (Commit)

Fungua faili la `Contributors.md`, shuka hadi mwisho wa ukurasa na
ongeza jina lako kisha hifadhi.

Mfano:

[James Smith](https://github.com/jamessmith)

Andika ujumbe:\
`Add <jina-lako> to Contributors list`

Kisha bonyeza `Commit to add-your-name`.

------------------------------------------------------------------------

## Tuma mabadiliko kwenda GitHub (Push)

Ingia kwenye Github.com kupitia GitHub Desktop kisha bonyeza `Publish`.

------------------------------------------------------------------------

## Wasilisha Pull Request

Nenda kwenye hifadhi yako GitHub kisha bonyeza
`Compare & pull request`.\
Wasilisha pull request yako.

Utapokea taarifa kupitia barua pepe mara mabadiliko yatakapounganishwa.

------------------------------------------------------------------------

## Uende wapi kuanzia hapa?

Hongera! Umefanikiwa kukamilisha mtiririko wa kawaida wa:

**fork → clone → edit → pull request**

Sherehekea mchango wako na ushiriki na wengine.
