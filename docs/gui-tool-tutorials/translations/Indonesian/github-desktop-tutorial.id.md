# Kontribusi Pertama (First Contributions)

| <img alt="GitHub Desktop" src="https://desktop.github.com/images/desktop-icon.svg" width="200"> | Edisi GitHub Desktop |
| ----------------------------------------------------------------------------------------------- | ---------------------- |

Memang sulit. Selalu terasa sulit ketika Anda melakukan sesuatu untuk pertama kalinya. Terutama saat Anda berkolaborasi, melakukan kesalahan bukanlah hal yang nyaman. Namun open source adalah tentang kolaborasi & bekerja sama. Kami ingin menyederhanakan cara kontributor open-source baru untuk belajar & berkontribusi untuk pertama kalinya.

Membaca artikel & menonton tutorial dapat membantu, tetapi apa yang lebih baik daripada benar-benar melakukan hal tersebut tanpa mengacaukan apa pun. Proyek ini bertujuan untuk memberikan panduan & menyederhanakan cara pemula melakukan kontribusi pertama mereka. Ingatlah bahwa semakin santai Anda, semakin baik Anda belajar. Jika Anda mencari cara untuk melakukan kontribusi pertama Anda, ikuti saja langkah-langkah sederhana di bawah ini. Kami berjanji, ini akan menyenangkan.

Jika Anda tidak memiliki GitHub Desktop di mesin Anda, [instal di sini](https://desktop.github.com/).

Jika Anda menggunakan versi GitHub desktop sebelum 1.0, [rujuk ke tutorial ini](github-desktop-old-version-tutorial.md).

<img align="right" width="300" src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/fork.png" alt="fork this repository" />

## Fork repository ini

Lakukan fork pada repo ini dengan mengklik tombol fork di kanan atas halaman ini.
Ini akan membuat salinan (copy) dari repository ini di akun Anda.

## Clone repository tersebut

Sekarang, lakukan clone pada repo ini ke mesin Anda.

PENTING: JANGAN CLONE REPO ASLINYA. Pergi ke fork Anda dan lakukan clone dari sana.

Untuk melakukan clone repo, klik "Code" dan kemudian klik "Open in Github Desktop".

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-clonetodesktop.png" alt="clone this repository" />

Sebuah jendela pop-up akan terbuka. Klik "Open GitHubDesktop.exe".

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-open-githubdesktop.png" alt="clone this repository" />

Setelah Anda mengklik "Open GitHubDesktop.exe", kotak dialog 'Clone a repository' akan muncul. Klik 'Clone'.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/clone-repository.png" alt="clone this repository" height="400" />  

Setelah itu akan muncul kotak dialog lain yang bertuliskan 'How are you planning to use this fork?' (Bagaimana Anda berencana menggunakan fork ini?). Pilih 'To contribute to the parent project' (Untuk berkontribusi pada proyek induk) dan klik 'Continue'.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/repository-clone-purpose.png" alt="Use of repository" height="500" />

Sekarang Anda telah menyalin isi dari repository `first-contributions` di GitHub ke komputer Anda.

## Buat branch baru

Sekarang buatlah sebuah branch dengan mengklik ikon "Current branch" di bagian atas lalu klik "New branch":

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch.png" alt="make a branch" />

Beri nama branch Anda dengan <tambahkan-nama-anda>. Contohnya, "add-james-smith"

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch-name.png" alt="name your branch" />

Klik `Create branch`

## Buat perubahan yang diperlukan dan commit perubahan tersebut

Sekarang, pergi ke tab *history* dan buka file `Contributors.md` di editor teks dengan mengklik kanan dan buka di editor teks (open in text editor). Gulir ke bagian bawah halaman dan tambahkan nama Anda ke dalamnya, lalu simpan file tersebut.

Contoh: Jika nama Anda adalah James Smith, maka akan terlihat seperti ini.

\[James Smith](https://github.com/jamessmith)

Anda dapat melihat bahwa ada perubahan pada file `Contributors.md` dan perubahan tersebut telah ditambahkan ke Github Desktop.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-status.png" alt="check status" />

Sekarang lakukan commit pada perubahan tersebut:

Tulis pesan "Add `<nama-anda>` to Contributors list" di kolom *summary*.

Ganti `<nama-anda>` dengan nama Anda yang sebenarnya.

Klik tombol bertuliskan `Commit to add-your-name`.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit1.png" alt="commit your changes" />

Di bagian bawah, Anda dapat melihat bahwa commit telah berhasil dibuat.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit2.png" alt="commit your changes" />

## Push perubahan ke GitHub

Klik File->Options dan masuk (sign-in) ke Github.com. Ketik username dan password Github Anda.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-sign-in.png" alt="log-in to Github" />

Klik tombol `Publish` di kanan atas.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-publish1.png" alt="push your changes" />

## Ajukan perubahan Anda untuk direview (Pull Request)

Jika Anda pergi ke repository Anda di GitHub, Anda akan melihat tombol `Compare & pull request`. Klik tombol tersebut.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/compare-and-pull.png" alt="create a pull request" />

Sekarang kirimkan (submit) pull request tersebut.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/submit-pull-request.png" alt="submit pull request" />

Segera saya akan menggabungkan (merge) semua perubahan Anda ke dalam master branch proyek ini. Anda akan mendapatkan email pemberitahuan setelah perubahan digabungkan.

## Ke mana selanjutnya?

Selamat! Anda baru saja menyelesaikan alur kerja standar _fork -> clone -> edit -> PR_ yang akan sering Anda temui sebagai kontributor!

Rayakan kontribusi Anda dan bagikan dengan teman dan pengikut Anda dengan mengunjungi [web app](https://firstcontributions.github.io#social-share).
