# Michango ya Kwanza

| <img alt="GitHub Desktop" src="https://desktop.github.com/images/desktop-icon.svg" width="200"> | GitHub Desktop Edition |
| ----------------------------------------------------------------------------------------------- | ---------------------- |

Ni ngumu. Ni ngumu kila wakati unapofanya kitu kwa mara ya kwanza. Haswa unaposhiriki, kufanya makosa sio kitu chepesi. Lakini chanzo huria kwa kawaida ni kuhusu kushiriki na kufanya kazi pamoja. Tulitaka kurahisisha njia ambayo wachangiaji wapya wa chanzo huria wanajifunza na kuchangia kwa mara ya kwanza.

Kusoma makala na kutazama vipindi vya mafunzo vinaweza kusaidia, lakini bora kuliko kufanya mambo halisi bila kuchanganya. Mradi huu unalenga kutoa mwongozo na kurahisisha njia ambayo wapya wanafanya michango yao ya kwanza. Kumbuka unapoweza kujisikia vizuri, utajifunza vyema. Ikiwa unatafuta kufanya michango yako ya kwa mara ya kwanza, fuata hatua rahisi hapa chini. Tunakuahidi, itakuwa ya kujifurahisha.

Ikiwa huna GitHub Desktop kwenye kompyuta yako, [sakinisha](https://desktop.github.com/).

Ikiwa unatumia toleo la GitHub desktop kabla ya 1.0, [tazama mafunzo haya](github-desktop-old-version-tutorial.md).

<img align="right" width="300" src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/fork.png" alt="fork this repository" />

## Fork hii repository

Fanya fork ya repo hii kwa kubonyeza kitufe cha fork upande wa kulia wa ukurasa huu.
Hii itaunda nakala ya repository hii kwenye akaunti yako.

## Clone repository

Sasa clone repo hii kwenye mashine yako.

MUHIMU: USI-CLONE ORIGINAL REPO. Nenda kwenye fork yako na iclone.

Ili kukloni repo, bonyeza "Code" kisha bonyeza "Open in Github Desktop".

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-clonetodesktop.png" alt="clone this repository" />

Dirisha litafunguka. Bonyeza "Open GitHubDesktop.exe".

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-open-githubdesktop.png" alt="clone this repository" />

Baada ya kubonyeza "Open GitHubDesktop.exe", sanduku la 'Clone a repository' linaonewa. Bonyeza 'Clone'.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/clone-repository.png" alt="clone this repository" height="400" />

Kisha sanduku lingine linaonewa linalosema 'Unapanaji kutumia fork hii?'. Chagua 'To contribute to the parent project' na bonyeza 'Continue'.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/repository-clone-purpose.png" alt="Use of repository" height="500" />

Sasa umenakili maudhui ya first-contributions repository kwenye kompyuta yako.

## Unda tawi

Sasa unda tawi kwa kubonyea ikoni ya "Current branch" juu na kisha bonyeza "New branch":

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch.png" alt="make a branch" />

Taja tawi lako <add-your-name>. Kwa mfano, "add-james-smith"

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch-name.png" alt="name your branch" />

Bonyeza `Create branch`

## Fanya mabadiliko na commit

Sasa, nenda kwenye kichupo cha history na ufungue faili ya `Contributors.md` kwa kubonyeza kulia na kufungua kwenye text editor. Nguruma chini ya ukurasa na ongeza jina lako, kisha hifadhi faili.

Mfano: Ikiwa jina lako ni James Smith, itakuwa hivi.

\[James Smith](https://github.com/jamessmith)

Unaweza kuona kuna mabadiliko kwenye Contributors.md na yameongezwa kwenye GitHub Desktop.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-status.png" alt="check status" />

Sasa commit mabadiliko:

Andika ujumbe "Add `<your-name>` to Contributors list" kwenye _summary_ field.

Badilisha `<your-name>` na jina lako.

Bonyeza kitufe kinachosema `Commit to add-your-name`.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit1.png" alt="commit your changes" />

Chini, unaweza kuona commit imeundwa.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit2.png" alt="commit your changes" />

## Peli mabadiliko kwenye github

Bonyeza File->Options na ingia kwenye Github.com. Andika jina lako la mtumiaji na nenosiri lako.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-sign-in.png" alt="log-in to Github" />

Bonyeza kitufe cha `Publish` upande wa kulia.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-publish1.png" alt="push your changes" />

## Wasilisha mabadiliko yako kwa ukaguzi

Ikiwa unaenda kwenye repository yako kwenye github, utaona kitufe cha `Compare & pull request`. Bonyeza kitufe hicho.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/compare-and-pull.png" alt="create a pull request" />

Sasa wasilisha pull request.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/submit-pull-request.png" alt="submit pull request" />

Hivi karibuni nitachangua mabadiliko yako yote kwenye tawi la master la mradi huu. Utapata arifa ya barua pepe mara mabadiliko yatakapochangwa.

## Kwenda wapi sasa?

Hongera! Umefanya kazi ya kawaida ya _fork -> clone -> edit -> PR_ ambayo utakutana nayo mara nyingi kama mchangiaji!

Adhimisha michango yako na ushiriki na marafiki na wafuataji kwa kwenda kwenye [tovuti](https://firstcontributions.github.io#social-share).
