# முதல் பங்களாப்பு

| <img alt="GitHub Desktop" src="https://desktop.github.com/images/desktop-icon.svg" width="200"> | GitHub Desktop பதிப்பு |
| ----------------------------------------------------------------------------------------------- | ---------------------- |

இது கடினமாக உள்ளது. நீங்கள் முதல் முறை ஏதாவது செய்யும்போது இது எப்போதும் கடினமாக இருக்கும். குறிப்பாக நீங்கள் ஒத்துழைக்கும்போது, பிழைகளை செய்வது ஒரு வசதியான விஷயம் அல்ல. ஆனால் திறந்த மூல குறிப்பு ஒত்துழைப்பு மற்றும் ஒருவருடன் வேலை செய்வது பற்றியது. புதிய திறந்த மூல பங்களாபாளர்கள் கற்பிக்கும் மற்றும் முதல் முறை பங்களாபவும் செய்ய வேண்டிய வழியை எளிமையாக்க விரும்பினோம்.

கட்டுரைகளைப் படிப்பது மற்றும் வேலிப் பயிற்சிகளைப் பார்ப்பது உதவ முடியும், ஆனால் எந்த விஷயத்தையும் செய்ய முடிவதை விட சிறந்ததாக எதுவும் இல்லை. இந்த திட்டம் வழிகாட்டுதலை வழங்கவும் ஆரம்பநிலையாளர்கள் தங்கள் முதல் பங்களாப்பைச் செய்யும் வழியை எளிமையாக்கவும் நோக்கம் கொண்டுள்ளது. நீங்கள் எவ்வளவு தளர்ந்திருக்கிறீர்களோ, அவ்வளவு நன்கு கற்கிறீர்கள் என்பதை நினைவில் வைக்கவும். உங்கள் முதல் பங்களாப்பை செய்ய விரும்பினால், கீழே உள்ள எளிய படிகளைப் பின்பற்றவும். நாங்கள் உங்களுக்கு வாக்குறுதி கொடுக்கிறோம், இது재미있을் என்று.

உங்கள் இயந்திரத்தில் GitHub Desktop இல்லாவிட்டால், [அதை நிறுவவும்](https://desktop.github.com/).

நீங்கள் GitHub desktop 1.0 க்கு முன்பு பதிப்பைப் பயன்படுத்துகிறீர்கள் என்றால், [இந்த பயிற்சியைக் குறிப்பிடவும்](github-desktop-old-version-tutorial.md).

<img align="right" width="300" src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/fork.png" alt="இந்த களஞ்சியத்தை கிளை பிரிக்கவும்" />

## இந்த களஞ்சியத்தை கிளை பிரிக்கவும்

இந்த இந்தக்கு சிறுகு பொத்தான்களை இந்த பக்கத்தின் மேல் வலதுபக்கத்தில் கிளிக் செய்வதன் மூலம் கிளை பிரிக்கவும்.
இது உங்கள் கணக்கில் இந்த களஞ்சியத்தின் நகலை உருவாக்கும்.

## களஞ்சியத்தை நকலாக்கவும்

இப்போது இந்த repos நுக்கலுக்கு உங்கள் இயந்திரத்திற்கு நகலாக்கவும்.

முக்கியமான: அசல் repos ஐ நகலாக்க வேண்டாம். உங்கள் கிளை பிரிக்கப்பட்ட பிரান்சுக்கு சென்று அதை நகலாக்கவும்.

repos ஐ நகலாக்க, "Code" மீது கிளிக் செய்து, பின்னர் "Github Desktop இல் திறக்கவும்" மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-clonetodesktop.png" alt="இந்த களஞ்சியத்தை நகலாக்கவும்" />

ஒரு பாப் அப் சாளரம் திறக்கப்படும். "Open GitHubDesktop.exe" மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-open-githubdesktop.png" alt="இந்த களஞ்சியத்தை நகலாக்கவும்" />

நீங்கள் "Open GitHubDesktop.exe" மீது கிளிக் செய்த பிறகு, 'களஞ்சியத்தை நகலாக்கவும்' உரையாடல் பெட்டி தோன்றும். 'நகலாக்கவும்' மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/clone-repository.png" alt="இந்த களஞ்சியத்தை நகலாக்கவும்" height="400" />  

அதற்குப் பிறகு 'இந்த கிளை பிரிக்கப்பட்ட பக்கத்தை நீங்கள் எவ்வாறு பயன்படுத்த திட்டமிட்டுள்ளீர்கள்?' என்று கூறும் மற்றொரு உரையாடல் பெட்டி தோன்றும். 'பெற்றோர் திட்டத்தில் பங்களாப்பு செய்ய' தேர்வு செய்து 'தொடரவும்' மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/repository-clone-purpose.png" alt="களஞ்சியத்தின் பயன்" height="500" />

இப்போது நீங்கள் github இலிருந்து first-contributions களஞ்சியத்தின் உள்ளடக்கத்தை உங்கள் கணினியில் நகலாக்கி உள்ளீர்கள்.

## ஒரு கிளை உருவாக்கவும்

இப்போது மேல்பகுதியில் "தற்போதைய கிளை" ஐகானை கிளிக் செய்து, பின்னர் "புதிய கிளை" மீது கிளிக் செய்வதன் மூலம் ஒரு கிளை உருவாக்கவும்:

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch.png" alt="ஒரு கிளை உருவாக்கவும்" />

உங்கள் கிளைக்கு <உங்கள்-பெயர் சேர்க்கவும்> என்ற பெயரை கொடுக்கவும். எடுத்துக்காட்டாக, "add-james-smith"

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-create-branch-name.png" alt="உங்கள் கிளைக்கு பெயரை கொடுக்கவும்" />

`கிளையை உருவாக்கவும்` மீது கிளிக் செய்யவும்

## அவசிய மாற்றங்களை செய்து அந்த மாற்றங்களை செயல்படுத்தவும்

இப்போது, ​​வரலாற்றுத் தாவலுக்குச் சென்று `Contributors.md` கோப்பை உரை சম்பাதகத்தில் திறக்க வலது கிளிக் செய்யவும். பக்கத்தின் கீழே உருட்டவும் மற்றும் உங்கள் பெயரைச் சேர்க்கவும், பின்னர் கோப்பை சேமிக்கவும்.

உதாரணம்: உங்கள் பெயர் James Smith என்றால், இது இப்படி இருக்க வேண்டும்.

\[James Smith](https://github.com/jamessmith)

Contributors.md இல் மாற்றங்கள் உள்ளதை நீங்கள் பார்க்கலாம் மற்றும் அவை Github Desktop க்கு சேர்க்கப்பட்டுள்ளன.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-status.png" alt="நிலையை சரிபார்க்கவும்" />

இப்போது அந்த மாற்றங்களை செயல்படுத்தவும்:

_summary_ புலத்தில் "உங்கள்-பெயரைச் சேர்க்கவும் பங்களாபாளர் பட்டியலிற்கு" என்ற செய்தியை எழுதவும்.

`<உங்கள்-பெயர்>` அ உங்கள் பெயரை மாற்றவும்.

`உங்கள்-பெயர்ஐ சேர்க்க செயல்படுத்தவும்` என்று கூறும் பொத்தான் மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit1.png" alt="உங்கள் மாற்றங்களை செயல்படுத்தவும்" />

கீழ்பகுதியில், செயல்படுத்துதல் உருவாக்கப்பட்டுள்ளது என்பதை நீங்கள் பார்க்கலாம்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-commit2.png" alt="உங்கள் மாற்றங்களை செயல்படுத்தவும்" />

## github க்கு மாற்றங்களை தாக்குதல் செய்யவும்

File->Options மீது கிளிக் செய்து Github.com இல் உள்நுழையவும். உங்கள் Github ஆணையை மற்றும் கடவுளை தட்டச்சு செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-sign-in.png" alt="Github இல் உள்நுழையவும்" />

மேல் வலதுபக்கத்தில் `வெளியீடு` பொத்தான் மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/dt1-publish1.png" alt="உங்கள் மாற்றங்களை தாக்குதல் செய்யவும்" />

## மறுஆய்வுக்கு உங்கள் மாற்றங்களை சமர்ப்பிக்கவும்

நீங்கள் github இல் உங்கள் களஞ்சியத்திற்குச் சென்றால், `ஒப்பிடுக மற்றும் இழுக் கோரிக்கை` பொத்தான் கண்ணுக்குத் தெரிவது. அந்த பொத்தான் மீது கிளிக் செய்யவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/compare-and-pull.png" alt="ஒரு இழுக் கோரிக்கையை உருவாக்கவும்" />

இப்போது இழுக் கோரிக்கையை சமர்ப்பிக்கவும்.

<img src="https://firstcontributions.github.io/assets/gui-tool-tutorials/github-desktop-tutorial/submit-pull-request.png" alt="இழுக் கோரிக்கையை சமர்ப்பிக்கவும்" />

விரைவில் நான் உங்கள் அனைத்து மாற்றங்களையும் இந்த திட்டத்தின் master கிளைக்கு இணைக்கிறேன். மாற்றங்கள் இணைக்கப்பட்ட பிறகு நீங்கள் ஒரு அறிவிப்பு மின்னஞ்சல் பெறுவீர்கள்.

## இப்போது நாம் எங்கே செல்லலாம்?

வாழ்த்துக்கள்! நீங்கள் நிலையான _கிளை பிரிப்பு -> நகலாக்குதல் -> திருத்தம் -> PR_ பணிப்பாய்வை இப்போது முடிந்துவிட்டீர்கள், இது ஒரு பங்களாபாளராக அடிக்கடி சந்திக்கப்படும்!

உங்கள் பங்களாப்பை கொண்டாடி [வெப் பயன்பாடு](https://firstcontributions.github.io#social-share) க்குச் சென்று உங்கள் நண்பர்கள் மற்றும் பின்பற்றுபவர்களுடன் பகிர்ந்து கொள்ளவும்.
