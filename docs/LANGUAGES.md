## 🤝 Contributing Translations

We're always looking for help! To contribute a translation:

1. **Choose a language** - Check if it's already in the list
2. **Fork the repository**
3. **Create or update** `docs/translations/README.[language-code].md`
4. **Submit a Pull Request** with your translation
5. **Join the Discussion** for your language
