# Додатни материјали

Претпостављамо да сте завршили основни туторијал пре него што сте дошли овде. Овај документ ће вам дати додатне информације о напредним  Git техникама.

### [Измена (енг. amending) commit-а](amending-a-commit.sr-Cyrl.md)
Овај документ пружа информације о томе како да измените commit на удаљеном репозиторијуму. Измена commit-а је начин да промените последњи commit који сте направили на тренутној грани. Ово може бити од помоћи када желите да промените commit поруку или ако сте заборавили да укључите неке измене у commit. Можете наставити да мењате commit док га не push-ујете на удаљени репозиторијум.
> Користите ово ако је потребно да прилагодите commit који сте направили.

### [Конфигурисање git-а](configuring-git.sr-Cyrl.md)
Овај документ пружа информације о томе како да конфигуришете корисничке податке и остале опције на git-у.
> Користите ово да боље контролишете вашу git конфигурацију.

### [Синхронизација вашег fork-а са репозиторијумом](keeping-your-fork-synced-with-this-repository.sr-Cyrl.md)
Овај документ пружа информације о томе како да синхронизујете ваш fork-овани репозиторијум са оригиналним репозиторијумом. Ово је важно, јер се надамо да ћете ви и многи други допринети пројекту.
> Пратите ове кораке ако ваш fork нема никакве промене из родитељског репозиторијума.

### [Померање commit-а на другу грану](moving-a-commit-to-a-different-branch.sr-Cyrl.md)
Овај документ пружа информације о томе како да померите commit на другу грану.
> Пратите ове кораке да бисте померили commit на другу грану.

### [Уклањање фајла](removing-a-file.sr-Cyrl.md)
Овај документ пружа информације о томе како да уклоните фајл са вашег локалног репозиторијума.
> Пратите ове кораке да бисте научили како да уклоните фајл пре commit-а.

### [Уклањање гране са вашег репозиторијума](removing-branch-from-your-repository.sr-Cyrl.md)
Овај документ пружа информације о томе како да обришете грану са вашег репозиторијума.
> Тек када ваш pull request буде спојен, пратите ове кораке.

### [Решавање merge конфликата](resolving-merge-conflicts.sr-Cyrl.md)
Овај документ пружа информације о томе како да решите merge конфликте.
> Пратите ове кораке да бисте разрешили досадне merge конфликте.

### [Враћање (енг. reverting) commit-а](reverting-a-commit.sr-Cyrl.md)
Овај документ пружа информације о томе како да вратите commit са удаљеног репозиторијума. Може бити корисно у случају да требате поништити commit који је већ push-ован на Github.
> Пратите ове кораке ако желите да вратите commit.

### [Squash-овање commit-ова](squashing-commits.sr-Cyrl.md)
Овај документ пружа информације о томе како да squash-ујете commit-ове уз интерактивни rebase.
> Користите ово када желите отворити PR на open source пројекту, а reviewer вам тражи да squash-ујете све commit-ове у један, са информативном commit поруком.

### [Поништавање локалног commit-а](undoing-a-commit.sr-Cyrl.md)
Овај документ пружа информације о томе како да поништите commit на вашем локалном репозиторијуму. Ово требате урадити када мислите да сте скршили ваш локални репозиторијум и желите да га ресетујете.
> Пратите ове кораке ако желите да поништите или ресетујете локални commit.

### [Корисни линкови](Useful-links-for-further-learning.sr-Cyrl.md)
Овај документ је посвећен сајтовима са саветима и триковима, блоговима и корисним сајтовима који чине наше животе лакшим. Садржи одличне референце да задовољи све наше потребе, било да смо почетници или експерти. Ова страница би требало да служи као индекс свих тих корисних линкова који могу помоћи свакоме ко је нов у open-source домену, али и онима који желе још више да науче.

### [Креирање .gitignore фајла](creating-a-gitignore-file.sr-Cyrl.md)
Овај документ објашњава шта .gitignore фајл ради, зашто га користити и како креирати .gitignore фајл. Овај фајл се користи у скоро свим git пројектима. Помаже да се commit-ују само неопходни фајлови на git.

### [Чување креденцијала](storing-credentials.sr-Cyrl.md)
Овај документ објашњава како да чувате ваше креденцијале за репозиторијуме. Ово може бити безбедоносни проблем, тако да вас молимо да пратите сигурносне полисе вашег радног места или студија.
