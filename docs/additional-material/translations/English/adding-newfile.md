1. Create a new file
Go to your project folder.
Use your preferred text editor or IDE to create a new file.
Give the file a proper name and save it.
2. Stage the file
Open the terminal and navigate to your repository folder.
Use the following command to stage the new file:
git add new_file.extension
3. Commit the file
After staging the file, create a commit.
Use the following command:
git commit -m "Added new file"
4. Push to remote repository
Your file is now in your local repository.
To send it to the remote repository, use:
git push remote_branch
Here, "remote_branch" is the name of the branch where you want to add the file.
