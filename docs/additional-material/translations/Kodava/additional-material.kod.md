# ಹೆಚ್ಚುವರಿ ಮಾಹಿತಿ

ನಾಂಗ ನಿಂಗ ಮೂಲಭೂತ ಟ್ಯುಟೋರಿಯಲ್ ಅನ್ನು ಮುಗಿಸಿರ ಎಂದು ಎಣಿಸುವ. ಹೆಚ್ಚುವರಿ ಮಾಹಿತಿಯು Git ನ ಹೆಚ್ಚಿನ ತಂತ್ರಗಳ ಕುರಿತು ವಿವರಗಳ್ ತಾವ.

### [ಒಂದು Commit ಪರಿಷ್ಕರಿಸುವದು](../git_workflow_scenarios/amending-a-commit.md)  
ಈ ದಸ್ತಾವೇಜು ರಿಮೋಟ್ Repository ಲ್ಲಿ ಒಂದು Commit ಅನ್ನು ಪರಿಷ್ಕರಿಸುವ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ನಿಂಗ ಆಗಲೇ ಮಾಡಿರುವ Commit ಅನ್ನು ಸರಿಪಡಿಸುವಕ್ ಈ ವಿಧಾನವನ್ನು ಬಳಸಿ.  

### [Git ಸಂರಚನೆ](../git_workflow_scenarios/configuring-git.md)  
ಈ ದಸ್ತಾವೇಜು ಬಳಕೆದಾರ ವಿವರಗಳ್ ಮತ್ತು ಇತರ Git ಆಯ್ಕೆಗಳ್ ಹೇಗೆ ಸಂರಚಿಸೋಕು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ನಿಮ್ಮ Git ಸೆಟ್ಟಿಂಗ್ಸ್‌ಗಳ್ ಉತ್ತಮವಾಗಿ ನಿಯಂತ್ರಿಸುವಕ್ ಇದನ್ನು ಬಳಸಿ.  

### [ನಿಮ್ಮ Fork ಅನ್ನು ಮೂಲ Repository ಯೊಂದಿಗೆ ಸಿಂಕ್ ಮಾಡುವುದು](../git_workflow_scenarios/keeping-your-fork-synced-with-this-repository.md)  
ಈ ದಸ್ತಾವೇಜು ನಿಮ್ಮ Fork ಅನ್ನು ಬೇಸ್ Repository ಯೊಂದಿಗೆ ನವೀಕರಿಸುವ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ. ಇದು ಮುಖ್ಯವು ಏಕೆಂದರೆ ನಿಂಗ ಮತ್ತು ಅನೇಕ ಜನ ಈ ಪ್ರಾಜೆಕ್ಟ್‌ಗೆ ಕೊಡುಗೆ ತಾಪ.  
> ನಿಮ್ಮ Fork ಮೂಲ Repository ಯ ಹೊಸ ಬದಲಾವಣೆಗಳ್ ಹೊಂದಿಲ್ಲದಿದ್ದರೆ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [ಒಂದು Commit ಅನ್ನು ಬೇರೆ Branch ಗೆ ಸ್ಥಳಾಂತರಿಸುವುದು](../git_workflow_scenarios/moving-a-commit-to-a-different-branch.md)  
ಈ ದಸ್ತಾವೇಜು ಒಂದು Commit ಅನ್ನು ಬೇರೆ Branch ಗೆ ಹೇಗೆ ಸ್ಥಳಾಂತರಿಸೋಕು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ನಿಂಗ ಒಂದು Commit ಅನ್ನು ಬೇರೆ Branch ಗೆ ಸ್ಥಳಾಂತರಿಸುವಕ್ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [ಒಂದು ಕಡತವನ್ನು ತೆಗೆಯುವದು](../git_workflow_scenarios/removing-a-file.md)  
ಈ ದಸ್ತಾವೇಜು ನಿಮ್ಮ ಸ್ಥಳೀಯ Repository ಯಿಂದ ಒಂದು ಕಡತವನ್ನು ಹೇಗೆ ತೆಗೆಯುವದು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ನಿಮ್ಮ ಸ್ಥಳೀಯ Repository ಯಿಂದ ಒಂದು ಕಡತವನ್ನು ತೆಗೆಯುವಕ್ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [Branch ಅನ್ನು ನಿಮ್ಮ Repository ಯಿಂದ ತೆಗೆಯುವದು](../git_workflow_scenarios/removing-branch-from-your-repository.md)  
ಈ ದಸ್ತಾವೇಜು ನಿಮ್ಮ Repository ಯಿಂದ ಒಂದು Branch ಅನ್ನು ಹೇಗೆ ಅಳಿಸೋಕು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ನಿಮ್ಮ Pull Request ಅನ್ನು ಮರ್ಜ್ ಮಾಡಿದ ಪಿನ್ನೆ ಮಾತ್ರ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [ಮರ್ಜ್ ಸಂಘರ್ಷಗಳ್ ಪರಿಹರಿಸುವದು](../git_workflow_scenarios/resolving-merge-conflicts.md)  
ಈ ದಸ್ತಾವೇಜು ಮರ್ಜ್ ಸಂಘರ್ಷಗಳ್ ಹೇಗೆ ಪರಿಹರಿಸೋಕು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ.  
> ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ ಅನಾವಶ್ಯಕ ಮರ್ಜ್ ಸಂಘರ್ಷಗಳ್ ಪರಿಹರಿಸಲು.  

### [ಒಂದು Commit ಅನ್ನು ಹಿಂತೆಗೆಯುವದು](../git_workflow_scenarios/reverting-a-commit.md)  
ಈ ದಸ್ತಾವೇಜು ರಿಮೋಟ್ Repository ಲ್ಲಿ ಮಾಡಿದ ಒಂದು Commit ಅನ್ನು ಹೇಗೆ ಹಿಂತೆಗೆಯುವದು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾಹಿತಿ ತಾವ. ಇದು GitHub ನಲ್ಲಿ ಪ್ರಕಟಿಸಿರುವ Commit ಅನ್ನು ಹಿಂತೆಗೆಯುವಕ್ ತುಂಬಾ ಉಪಯುಕ್ತ ಉಂಡು.  
> ನಿಂಗ ಒಂದು Commit ಅನ್ನು ಹಿಂತೆಗೆಯುವಕ್ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [Commit ಗಳ್ ಒಟ್ಟುಗೂಡಿಸುವದು](../git_workflow_scenarios/squashing-commits.md)  
ಈ ದಸ್ತಾವೇಜು ಬಹು Commit ಗಳ್ ಒಂದರೊಂದಿಗೆ ಒಟ್ಟುಗೂಡಿಸುವ (rebase) ವಿಧಾನವನ್ನು ವಿವರಿಸುವ.  
> ನಿಂಗ open-source ಪ್ರಾಜೆಕ್ಟ್‌ಗೆ PR ಸಲ್ಲಿಸಿದಾಗ ಮತ್ತು ವಿಮರ್ಶಕ ಎಲ್ಲಾ Commit ಗಳ್ ಒಂದರೊಂದಿಗೆ ಸಂಯೋಜಿಸುವಕ್ ಕೋರಿದರೆ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [ಸ್ಥಳೀಯ Commit ಅನ್ನು ಹಿಂತೆಗೆಯುವದು](../git_workflow_scenarios/undoing-a-commit.md)  
ಈ ದಸ್ತಾವೇಜು ನಿಮ್ಮ ಸ್ಥಳೀಯ Repository ಲ್ಲಿ Commit ಅನ್ನು ಹೇಗೆ ಹಿಂತೆಗೆಯುವದು ಎಂಬುದರ ಬಗ್ಗೆ ವಿವರಿಸುವ. ನಿಂಗ ತಪ್ಪು ಮಾಡಿದಂತೆ ಅನಿಸಿದಾಗ ಮತ್ತು ಅದನ್ನು ಹಿಂತೆಗೆಯುವಕ್ ಬಯಸಿದರೆ ಇದು ಉಪಯುಕ್ತ ಉಂಡು.  
> ನಿಂಗ ಸ್ಥಳೀಯ Commit ಅನ್ನು ಹಿಂತೆಗೆಯುವಕ್/ಮರುಹೊಂದಿಸುವಕ್ ಈ ಹಂತಗಳ್ ಅನುಸರಿಸಿ.  

### [ಉಪಯುಕ್ತ ಲಿಂಕ್‌ಗಳ್](../git_workflow_scenarios/Useful-links-for-further-learning.md)  
ಈ ದಸ್ತಾವೇಜು ಉಪಯುಕ್ತ ಬ್ಲಾಗ್‌ಗಳ್, ಪೋಸ್ಟ್‌ಗಳ್, ವೆಬ್‌ಸೈಟ್‌ಗಳ್, ಸಲಹೆಗಳ್ ಮತ್ತು ಉಪಾಯಗಳಿಗೆ ಮೀಸಲಾಗಿ ಉಂಡು, ಇದು ನಡ ಜೀವನವನ್ನು ಸುಲಭ ಮಾಡುವ. ನಿಂಗ ತಜ್ಞರಾಗಿರಲಿ ಅಥವಾ ಪ್ರಾರಂಭಿಕರಾಗಿರಲಿ, ಈ ಪುಟ open-source ಪ್ರಾಜೆಕ್ಟ್‌ಗಳ ಹೊಸ ಬಳಕೆದಾರರಿಗೆ ಅಥವಾ ಇನ್ನಷ್ಟು ಕಲಿಯುವಕ್ ಇಚ್ಛಿಸುವವರಿಗೆ ಉಪಯುಕ್ತವಾದ ಲಿಂಕ್‌ಗಳ ಸೂಚಿಯಾಗಿ ಕಾಂಬ.
