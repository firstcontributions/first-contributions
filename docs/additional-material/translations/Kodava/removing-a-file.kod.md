# Git ನಿಂದ ಫೈಲ್ ಅನ್ನು ತೆಗೆಯುವದು

ಕಾಲಕಾಲಕ್ಕೆ, Git ನಿಂದ ಫೈಲ್ ಅನ್ನು ತೆಗೆಯೋಕು, ಆದರೆ ಅದನ್ನು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್ ನಿಂದ ಅಳಿಸಬಾರದು. ಇದನ್ನು ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯ ಮೂಲಕ ಮಾಡಲಕ್ಕು:

```bash
git rm <file> --cached
```

## ಆಗಿದ್ದು ಏನ್?

Git ಫೈಲ್ ಅನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡ್ವದು ನಿಲ್ಲಿಸುವ. Git ಗೆ, ಅದು ಅಳಿಸಿದಂತೆ ಕಾಂಬ, ಆದರೆ ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್ ನಲ್ಲಿ ಅದು ಇನ್ನೂ ಇಪ್ಪ.

**ಗಮನಿಸಿ:** `--cached` ಆಯ್ಕೆಯನ್ನು ಬಳಸಿದರೆ ಮಾತ್ರ ಫೈಲ್ ಅನ್ನು Git ನಿಂದ ತೆಗೆದು, ಆದರೆ ನಿಮ್ಮ ಸಿಸ್ಟಮ್ ನಲ್ಲೇ ಇರಿಸಿಕೊಳ್ಳಲಕ್ಕು. ಈ ಆಯ್ಕೆಯನ್ನು ಬಳಸದೆ `git rm <file>` ಮಾಡಿದರೆ, ಅದು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್ ನಿಂದಲೂ ಫೈಲ್ ಅನ್ನು ಅಳಿಸಿಬಿಡುವ.

ನಿಂಗ ಈ ಬದಲಾವಣೆಯನ್ನು Commit ಮಾಡಿ, `git push origin master` ಬಳಸಿದರೆ, ಫೈಲ್ ನಿಮ್ಮ ರಿಮೋಟ್ Repository ಯಿಂದಲೂ ಅಳಿಸಿಬಿಡುವ.

## ಹೆಚ್ಚುವರಿ ವಿಶೇಷತೆಗಳ್

- ಹಲವಾರು ಫೈಲ್‌ಗಳ್ ಒಂದೇ ಆಜ್ಞೆಯಲ್ಲಿ ತೆಗೆಯುವಕ್:

    ```bash
    git rm file1.js file2.js file3.js --cached
    ```

- ಫೈಲ್ ಮಾದರಿಗಳ್ ಬಳಸಿ ಫೈಲ್‌ಗಳ್ ತೆಗೆಯುವಕ್:
  
    ```bash
    git rm *.txt --cached  # ಎಲ್ಲಾ .txt ಫೈಲ್‌ಗಳ್ Git ನಿಂದ ತೆಗೆಯುವ.
    ```
