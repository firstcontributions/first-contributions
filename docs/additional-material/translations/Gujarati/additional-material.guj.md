# વધારાની માહિતી

અમે ધારીએ છીએ કે તમે અહીં આવતા પહેલા મૂળભૂત ટ્યુટોરિયલ સાથે પહેલેથી જ પૂર્ણ કરી લીધું છે. આ દસ્તાવેજ તમને અદ્યતન Git તકનીકો વિશે કેટલીક વધારાની માહિતી આપશે.

### [કમિટમાં સુધારો કરવો](amending-a-commit.guj.md)
આ દસ્તાવેજ રિમોટ રિપોઝિટરી પર કમિટમાં સુધારો કેવી રીતે કરવો તે વિશે માહિતી પ્રદાન કરે છે.
> જ્યારે તમારે કરેલી કમિટને સમાયોજિત કરવાની જરૂર હોય ત્યારે આનો ઉપયોગ કરો.

### [Git ને કન્ફિગર કરવું](configuring-git.guj.md)
આ દસ્તાવેજ git માં વપરાશકર્તાની વિગતો અને અન્ય વિકલ્પોને કેવી રીતે ગોઠવવા તે વિશે માહિતી પ્રદાન કરે છે.
> તમારી git ગોઠવણીઓને વધુ સારી રીતે નિયંત્રિત કરવા માટે આનો ઉપયોગ કરો.

### [તમારા ફોર્કને રિપોઝિટરી સાથે સિંક રાખવું](keeping-your-fork-synced-with-this-repository.guj.md)
આ દસ્તાવેજ તમારા ફોર્ક કરેલા રિપોઝિટરીને બેઝ રિપોઝિટરી સાથે અપ-ટુ-ડેટ કેવી રીતે રાખવું તે વિશે માહિતી પ્રદાન કરે છે.
> જો તમારા ફોર્કમાં પેરેન્ટ રિપોઝિટરીમાં કોઈ ફેરફારો ન હોય તો આ પગલાંઓ અનુસરો.

### [કમિટને અલગ બ્રાન્ચમાં ખસેડવી](moving-a-commit-to-a-different-branch.guj.md)
આ દસ્તાવેજ કમિટને બીજી બ્રાન્ચમાં કેવી રીતે ખસેડવી તે વિશે માહિતી પ્રદાન કરે છે.
> કમિટને બીજી બ્રાન્ચમાં ખસેડવા માટે આ પગલાંઓ લો.

### [ફાઇલ દૂર કરવી](removing-a-file.guj.md)
આ દસ્તાવેજ તમારા લોકલ રિપોઝિટરીમાંથી ફાઇલ કેવી રીતે દૂર કરવી તે વિશે માહિતી પ્રદાન કરે છે.
> કમિટ પહેલાં ફાઇલ કેવી રીતે દૂર કરવી તે શીખવા માટે આ પગલાંઓ અનુસરો.

### [તમારા રિપોઝિટરીમાંથી બ્રાન્ચ દૂર કરવી](removing-branch-from-your-repository.guj.md)
આ દસ્તાવેજ તમારા રિપોઝિટરીમાંથી બ્રાન્ચ કેવી રીતે ડિલીટ કરવી તે વિશે માહિતી પ્રદાન કરે છે.
> તમારી પુલ રિક્વેસ્ટ મર્જ થયા પછી જ આગળના પગલાંઓ અનુસરો.

### [મર્જ કોન્ફ્લિક્ટ્સ ઉકેલવા](resolving-merge-conflicts.guj.md)
આ દસ્તાવેજ મર્જ કોન્ફ્લિક્ટ્સ કેવી રીતે ઉકેલવા તે વિશે માહિતી પ્રદાન કરે છે.
> હેરાન કરતા મર્જ કોન્ફ્લિક્ટ્સને ઉકેલવા માટે આ પગલાંઓ લો.

### [કમિટને રિવર્ટ કરવી](reverting-a-commit.guj.md)
આ દસ્તાવેજ રિમોટ રિપોઝિટરી પર કમિટને રિવર્ટ કેવી રીતે કરવી તે વિશે માહિતી પ્રદાન કરે છે.
> જો તમે કમિટને રિવર્સ કરવા માંગતા હો તો આ પગલાંઓ લો.

### [કમિટ્સને સ્ક્વોશ કરવી](squashing-commits.guj.md)
આ દસ્તાવેજ ઇન્ટરેક્ટિવ રિબેઝ સાથે કમિટ્સને સ્ક્વોશ કેવી રીતે કરવી તે વિશે માહિતી પ્રદાન કરે છે.
> જો તમે ઓપન સોર્સ પ્રોજેક્ટમાં PR ખોલવા માંગો છો અને રિવ્યુઅર તમને દરેક કમિટને એકમાં સ્ક્વોશ કરવાનું કહે તો આનો ઉપયોગ કરો.

### [લોકલ કમિટને અનડૂ કરવી](undoing-a-commit.guj.md)
આ દસ્તાવેજ તમારા લોકલ રિપોઝિટરી પર કમિટને અનડૂ કેવી રીતે કરવી તે વિશે માહિતી પ્રદાન કરે છે.
> જો તમે લોકલ કમિટને અનડૂ/રિસેટ કરવા માંગતા હો તો આ પગલાંઓ લો.

### [ઉપયોગી લિંક્સ](Useful-links-for-further-learning.guj.md)
આ દસ્તાવેજ તમામ ટિપ્સ અને ટ્રિક્સ વેબસાઇટ્સ, બ્લોગ પોસ્ટ્સ અને મદદરૂપ સાઇટ્સ માટે સમર્પિત છે જે આપણા જીવનને સરળ બનાવે છે.
