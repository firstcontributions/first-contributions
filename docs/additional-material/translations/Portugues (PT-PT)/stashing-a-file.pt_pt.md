# Guardar alterações temporariamente (stash)

Se tens alterações que não queres cometer ainda, podes guardá‑las temporariamente com `git stash`.

```
git stash          # guarda alterações
git stash list     # lista stashes
git stash pop      # aplica e remove o stash mais recente
```

Isto é útil para alternar de branch sem perder trabalho em progresso.
