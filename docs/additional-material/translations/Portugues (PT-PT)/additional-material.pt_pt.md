# Informações Adicionais

Presumimos que já tenhas terminado o tutorial básico antes de chegares aqui. Este documento dá‑te informações adicionais sobre técnicas avançadas do Git.

### [Emendar um commit](amending-a-commit.pt_pt.md)
Este documento explica como emendar o commit mais recente no repositório remoto. Emendar um commit permite alterar a mensagem do commit ou incluir alterações que tenham sido esquecidas. Podes emendar um commit quantas vezes quiseres antes de o enviares para o remoto.
> Usa isto quando precisares de corrigir um commit que já fizeste.

### [Configurar o git](confinguring-git.pt_pt.md)
Este documento explica como configurar os detalhes de utilizador e outras opções do `git`.
> Usa isto para controlar melhor as tuas configurações do `git`.

### [Manter o teu fork sincronizado com o repositório](keeping-your-fork-synced-with-this-repository.pt_pt.md)
Este documento explica como manter o teu fork atualizado em relação ao repositório base. Isto é importante pois espera‑se que tu e muitas outras pessoas contribuam para o projeto.
> Segue estes passos se o teu fork não incluir as alterações do repositório principal.

### [Mover um commit para outro branch](moving-a-commit-to-a-different-branch.pt_pt.md)
Este documento explica como mover um commit para outro branch.
> Segue estes passos para deslocar um commit para outro branch.

### [Remover um ficheiro](removing-a-file.pt_pt.md)
Este documento explica como remover um ficheiro do teu repositório local.
> Segue estes passos para aprender a remover um ficheiro antes de fazeres um commit.

### [Eliminar um branch do teu repositório](removing-branch-from-your-repository.pt_pt.md)
Este documento explica como apagar um branch do teu repositório.

### [Resolver conflitos de merge](resolving-merge-conflicts.pt_pt.md)

Este documento explica como reverter um commit no repositório remoto. É útil quando precisas de desfazer um commit que já foi enviado para o GitHub.
