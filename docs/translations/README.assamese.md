# প্ৰথম অৱদান (First Contributions)

এই প্ৰকল্পটো নতুন অৱদানকাৰীসকলক তেওঁলোকৰ প্ৰথম অৱদান আগবঢ়াবলৈ সহায় কৰিবলৈ তৈয়াৰ কৰা হৈছে।

যদি আপুনি অৱদান আগবঢ়াবলৈ বিচাৰে, তেন্তে তলত দিয়া পদক্ষেপসমূহ অনুসৰণ কৰক।

## প্ৰয়োজনীয়তা
আপোনাৰ কম্পিউটাৰত Git ইনস্টল থকাটো প্ৰয়োজন।

## পদক্ষেপসমূহ

১. এই প্ৰকল্পটো ফৰ্ক (Fork) কৰক।
২. এই প্ৰকল্পটো আপোনাৰ কম্পিউটাৰলৈ ক্ল'ন (Clone) কৰক।
৩. এটা নতুন ব্ৰাঞ্চ (Branch) তৈয়াৰ কৰক।
৪. আপোনাৰ পৰিৱৰ্তনসমূহ কৰক।
৫. আপোনাৰ পৰিৱৰ্তনসমূহ কমিট (Commit) আৰু পুশ্ব (Push) কৰক।
৬. এটা পুল ৰিকুৱেষ্ট (Pull Request) দাখিল কৰক।

আপুনি আপোনাৰ অৱদানৰ বাবে এটা পুল ৰিকুৱেষ্ট দাখিল কৰাৰ পিছত, আমি ইয়াক পৰীক্ষা কৰিম আৰু আপোনাৰ অৱদানক মূল প্ৰকল্পত অন্তৰ্ভুক্ত কৰিম।
