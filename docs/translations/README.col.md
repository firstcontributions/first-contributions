# Primeras Contribuciones

Uy, la cosa está complicada. La primera vez que uno se mete en algo nuevo suele ser difícil, ¿cierto? Más todavía cuando toca trabajar con otras personas, porque nadie quiere meter la pata y quedar mal. Queremos que a quienes están empezando en el código abierto se les haga más fácil aprender y aportar su granito de arena.

Está muy bien leer artículos y ver tutoriales, pero nada como aprender haciendo. Este proyecto es una guía sencilla para que los principiantes aprendan a contribuir por primera vez. Si quieres dar tus primeros pasos, sigue estos pasos con calma.

#### _Si todavía no te llevas bien con la consola, [aquí tienes tutoriales con herramientas más amigables (GUI)](#tutoriales-con-otras-herramientas)_

## Hazle un Fork a este repositorio

Haz clic en el botón **Fork** que está en la parte superior derecha de esta página.  
Esto creará una copia de este repositorio en tu cuenta de GitHub.

---

## Clona el repositorio forkeado

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/clone.png" alt="clonar este repositorio" />

Ahora necesitas clonar ese repositorio en tu computador. Ve a tu cuenta de GitHub, entra a tu fork del repositorio, haz clic en **Code** y luego en el icono para **copiar la URL al portapapeles**.

Abre la terminal o consola y ejecuta el siguiente comando:

```bash
git clone "la-url-que-acabaste-de-copiar"
```
