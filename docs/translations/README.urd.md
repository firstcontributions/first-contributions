# پہلی شراکت

یہ پراجیکٹ نئے لوگوں کے لیے ہے جو پہلی بار اوپن سورس میں حصہ ڈالنا چاہتے ہیں۔

اگر آپ کمانڈ لائن استعمال کرنا چاہتے ہیں تو یہ ٹیوٹوریل آپ کے لیے ہے۔

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/fork.png" alt="اس ریپو کو فورک کریں" />

## ضروریات

اگر آپ کے کمپیوٹر میں git نہیں ہے تو [یہاں سے انسٹال کریں](https://help.github.com/articles/set-up-git/)۔

## یہ ریپو فورک کریں

اوپر دائیں طرف Fork کے بٹن پر کلک کریں۔
اس سے آپ کے اکاؤنٹ میں اس ریپو کی کاپی بن جائے گی۔

## ریپو کلون کریں

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/clone.png" alt="اس ریپو کو کلون کریں" />

GitHub پر اپنے اکاؤنٹ میں جائیں، فورک کیا ہوا ریپو کھولیں اور Code کے بٹن پر کلک کریں، پھر copy آئیکن پر کلک کریں۔
