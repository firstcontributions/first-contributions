
First Contributions (Akaa a hiihii a eren)

Project ne ngu a kwagh u yilan ser a vande nan ior mba ve lu hiihii u eren kwagh u tesen ve er ve er "first contribution" yô. Aluer u ngu keren u eren "first contribution" yô, dondo akaa a sha shin ne.

Aluer u fa kwagh u "command line" ga yô, nenge sha u tesen u akaa a GUI sha shin ne.
Aluer u ngu a "git" sha computer wou ga yô, yila u (install).
Fork this repository (Nenge sha repository ne)

Fork this repository sha u kpen sha "fork" button u a lu sha heen ne. Kwagh ne una er kopi u repository ne sha account wou.
Clone the repository (Kura repository ne sha computer wou)

Saa, kura repository ne sha computer wou. Za sha GitHub account wou, bugh repository u u fork u er la, kpen sha "code" button la, maa kpen sha "SSH" tab, maa kpen sha icon u a lu "copy url" la.

Bugh terminal maa er git command ne sha shin ne:
Bash

git clone "url u u kopy u er la"

Create a branch (Er branch)

Za sha folder u repository la sha computer wou:
Bash

cd first-contributions

Maa er branch sha u eren git switch command:
Bash

git switch -c your-new-branch-name
