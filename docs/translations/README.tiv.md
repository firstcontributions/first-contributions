# Gbenda u Hiihii u Kpela (First Contributions)

Ka i taver kpishi u eren kwagh sha qua u hiihii. Mciem ma eren isholibo ka ma na i kera fatyô u eren kwagh ga, kpa hemban je yô shighe u i lu eren kwagh sha ishigh ki ior la. Kpa, gbenda u open source yô ka u gban zwa môm shi eren kwagh imôngo. Nahan se soo bende a ior mba ve lu a mfe ga la sha u tesen ve gbenda u a er kwagh hiihii la.

Ngeren man mtese kpaa ka a wase, kpa kanyi i hembe doon u eren kwagh sha u kôron un ave? Ikyar i project ne ka u wasen ior gbenda u dedoo man u pinen mbahembancin sha gbenda u a er kwagh la. Kôndon nen, ne mba a mfe u tseer tseer yô ne la. Cii yô u eren kwagh sha hiihii yô, er atindi ne sha ashighe a dondo ne.

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/fork.png" alt="kop repository ne" />

Ne a lu a git sha mter wen ga yô, [kama un sha](https://help.github.com/articles/set-up-git/).

## Kop agwa ne (Fork this repository)

Kop agwa ne sha u mian inya i "Fork" i lu sha igwa i page la.
Kwagh ne una yôô nambe wen u akun a GitHub i agwa ne.

## Dua agwa ne (Clone this repository)

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/clone.png" alt="dua agwa ne" />

Hii u dua agwa ne sha machine wen. Na "Clone" ina maa mian sha u *copy to clipboard* ikon.

Buugh zwa terminal u tsua git command ne:
```bash
git clone "url u i copy la"
```

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/copy-to-clipboard.png" alt="copy URL sha clipboard" />

U tesen:
```bash
git clone https://github.com/iti-you/first-contributions.git
```

## Gba nongo (Create a branch)

Tim a inya sha tsua repository folder (ne wea lu cii tsua ga yô):
```bash
cd first-contributions
```

Gba nongo u wain a `git checkout` command:
```bash
git checkout -b <iti-nongo-you>
```

U tesen:
```bash
git checkout -b add-tiv-translation
```

## Sôron man za a mi (Make necessary changes and commit those changes)

Buugh `Contributors.md` file man na iti you za a mi, nahan na save i file la.

<img align="right" width="450" src="https://firstcontributions.github.io/assets/Readme/git-status.png" alt="git status" />

Ne tese i `git status`, wea kpaa u nenge i change i er la. Za a change ne a branch u er la a `git add` command:
```bash
git add Contributors.md
```

Za a file ne a `git commit` command:
```bash
git commit -m "Na iti <iti-you> sha Contributors list"
```

## Tindi (Push changes to GitHub)

Tindi i change a `git push` command:
```bash
git push origin <iti-nongo-you>
```

## Gba na Mpin (Submit your changes for review)

Za sha repository wen sha GitHub, ne a nenge mian `Compare & pull request` inya.

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/compare-and-pull.png" alt="gba pull request" />

Nahan er tindi pull request wen.

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/submit-pull-request.png" alt="tindi pull request" />

M wase we dedoo! M a ungwa ne sha mkaanem la man ikyar u kwagh ne! 🎉
